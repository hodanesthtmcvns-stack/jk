# ICU Lab Trends & Alerts: Help & About

### Purpose
Transforms raw Laboratory Information System (LIS) data into a longitudinal trend matrix with automated clinical alerts.

### Automated Alerts
* **AKI:** Detects rises based on KDIGO-style criteria.
* **TLS:** Tracks Uric Acid, K+, Phosphorus, and Calcium.
* **DIC:** Calculates automated **ISTH score**.

### How to Use
1. **Save Data:** Save your LIS page as "Webpage, HTML only."
2. **Upload:** Use the "Choose File" button in the app.
3. **Export:** Use "Export PDF" for professional multi-day reports for ward rounds. Use "Excel" for professional quality multi-day reports for clinical audit and research data collection.

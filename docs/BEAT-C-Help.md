# BEAT-C: Help & About

### Purpose
Assists in selecting the "Best-fit" empiric antibiotics based on local epidemiology (TMC VNS 2024) and patient-specific risk factors.

### How to Use
1. **Patient Factors:** Toggle Shock, Hospital-onset, Neutropenia, or Renal Failure.
2. **Suspected Source:** Select Pneumonia, UTI, CNS, etc.
3. **Renal Dosing:** Enter SCr for auto-calculation of CrCl (Cockcroftâ€“Gault).
4. **Compute:** Review the top 3 regimens ranked by coverage percentage.

# Patient Label Generator: Help & About
**Lightweight Identification Tool for ICU & OT**

### Purpose
A specialized tool for rapid printing of standardized patient identification labels. It ensures that patient data is formatted into a precise grid layout compatible with standard A4 sticker sheets.

### How to Use
1. **Input Data:** Paste or type the patient information (Name, ID, Age/Sex, Ward) into the text area at the top.
2. **Generate:** Click **"Generate Labels"** to instantly fill all 40 cells on the virtual sheet.
3. **Print:**
   * Click the **"Print"** button.
   * **Crucial Setting:** In the browser print dialog, set **Paper Size to A4** and **Margins to NONE**. This ensures the text aligns perfectly with the stickers.

### Technical Specifications
* **Layout:** 4x10 grid (40 labels per A4 sheet).
* **Label Size:** Precisely 52.5mm x 29.7mm.
* **Privacy:** Operates entirely client-side; no patient data is ever uploaded or stored.
